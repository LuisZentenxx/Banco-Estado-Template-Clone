# Proyecto Frontend de Banco Estado
## Descripción

Este proyecto es parte de mi proceso de autoaprendizaje y tiene como objetivo proporcionar una implementación básica de la interfaz de usuario del sitio de Banco Estado. Es fundamental destacar que este proyecto es exclusivamente con fines educativos y no será utilizado con fines comerciales ni maliciosos.

## Sitio Oficial Banco Estado
Puedes visitar el sitio oficial de Banco Estado [aquí](https://www.bancoestado.cl/content/bancoestado-public/cl/es/home/home.html#/).

## Tecnologías Utilizadas

- HTML5
- Tailwind CSS
- JavaScript
